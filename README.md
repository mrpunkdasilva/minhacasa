## About the Project

**Minha Casa** is a sophisticated platform designed to provide a seamless experience for users looking for their next home. It focuses on a clean UI, smooth animations, and a modern aesthetic.

### Key Features

- **Modern UI/UX**: Built with React 19 and Tailwind CSS 4 for a fast and responsive interface.
- **Animated Components**: Features a custom-built, high-performance SVG animated logo and loading states.
- **Contextual Loading**: A dedicated loading experience that provides visual feedback to users while the app prepares.
- **Type Safety**: Fully implemented with TypeScript for better developer experience and reliability.

---

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Formatting**: [Prettier](https://prettier.io/)
- **Icons**: [Phosphor Icons](https://phosphoricons.com/)
- **Components**: [Radix UI](https://www.radix-ui.com/) & [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [tw-animate-css](https://github.com/leandrosimard/tw-animate-css)

## Getting Started

### Prerequisites

- Node.js (v18.0.0 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/minhacasa.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

```bash
npm run dev
```

The app will be available at [http://localhost:3000](http://localhost:3000).

### Building for Production

```bash
npm run build
npm run start
```
